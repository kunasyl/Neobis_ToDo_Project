## ToDoProject
**ToDoProject** - приложение для создания и отслеживания задач

### Структура
`/` - Главная страница
`/add` - Добавить задачу
`/update/id` - Сменить статус задачи
`/delete/id` - Удалить задание

### Установка
Проект написан на django framework. В `requirements.txt` указаны нужные пакеты.
